# -B-
这是一个面向抖音与 B 站直播场景的实时监控看板项目，用于按关键词持续抓取正在直播的作者与直播间数据，并以“节点”方式保存每次抓取结果。项目支持用户统一选择监控平台、配置抓取间隔、设置企业微信推送、维护作者主页黑名单，并对抖音 Cookie 进行管理。系统每次抓取后只推送当前节点数据，同时支持按照指定节点范围进行手动汇总与立即推送，满足跨时间段监控、运营复盘和日报汇总等需求。整体界面采用统一配置与统一操作流程，减少平台间重复配置，提升日常监控与汇总效率。  如果你要放到项目首页，也可以用这一版更短的描述：  基于 Flask 的抖音/B 站直播监控看板，支持关键词抓取、节点化数据保存、企业微信推送、节点区间汇总、黑名单过滤与统一平台配置，适用于直播监控、数据复盘与定时汇总场景。
